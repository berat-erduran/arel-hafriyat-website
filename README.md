# Arel Hafriyat Website

## Project Overview

Arel Hafriyat Website is a corporate web project developed to present the company's services, projects and contact information through a modern, user-friendly and responsive interface.

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Responsive Web Design

## Features

- Corporate company presentation
- Service detail pages
- Contact information page
- Mobile-friendly design
- Fast and lightweight structure
- User-oriented navigation

## Purpose

The purpose of this project is to provide visitors with easy access to company information and services while maintaining a professional digital presence.
